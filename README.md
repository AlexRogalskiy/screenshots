***Styled Screenshots*** is a serverless function that generates dynamically styled graph images based on SVG (Scalable Vector Graphics).
For the tech stack, ***Styled Screenshots*** using Typescript and serverless function from Vercel stack.

## *How to use*

It's simple, you can copy paste this markdown content, like this one:

```
![Styled Screenshots](https://styled-screenshots.vercel.app/api?url=[value]&width=[value]&height=[value]&fullPage=[value]&type=[value]&encoding=[value]&selector=[value])
```

There are several options you can use from the list:

|  **Options**     | **Description**            |   **Type**                        | **Example**             | **Query Params**      |
| ---------------- | -------------------------- | --------------------------------- | ----------------------- | --------------------- |
| **\[Url]**       | Json data source url      | <code>String</code>                | https://www.google.com/ | `?url=[value]`        |
| **\[Width]**     | Chart graph image width   | <code>Numeric</code>               | 400                     | `&width=[value]`      |
| **\[Height]**    | Chart graph image height  | <code>Numeric</code>               | 400                     | `&height=[value]`     |
| **\[FullPage]**  | Full page viewport        | <code>Boolean/numeric</code>       | true/false || 1/0       | `&fullPage=[value]`   |
| **\[Type]**      | Image content type        | <code>String</code>                | jpeg | png              | `&type=[value]`       |
| **\[Encoding]**  | Image encoding            | <code>String</code>                | base64 | binary         | `&encoding=[value]`   |
| **\[Selector]**  | Html element selector     | <code>String</code>                | #element                | `&selector=[value]`   |
